# Student Manager

A Flask web application to manage student records.

## Features
- Add new students
- View all students
- Edit student information
- Delete students

## Technologies
- Python
- Flask
- SQLite3
- HTML

## Run the App
```bash
pip install flask
python app.py

## 📁 My Files

- `app.py` - Main Flask application
- `database.sql` - Database structure/schema
- `templates/` - HTML template files
- `.gitignore` - Git ignore rules
- `README.md` - Project documentation
